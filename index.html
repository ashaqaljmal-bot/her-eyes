أمان API

الطلب يُرسل إلى API خارجية عبر endpoint خاص، مما يحفظ مفتاح Anthropic بعيداً عن الواجهة.

🌐

واجهة عربية متكاملة

تصميم RTL جاهز ودعم نص عربي واضح، مناسب للزوار من الشرق الأوسط وشمال أفريقيا.

🚀

نشر سريع

يعمل مباشرة بعد رفع المشروع إلى GitHub وربطه بـ Vercel.

كيف يعمل المشروع؟

الواجهة الأمامية في `public/index.html` تسمح للمستخدم بإرسال رسالة. الطلب يمر عبر `api/chat.js` الذي يتصل بـ Claude بواسطة `ANTHROPIC_API_KEY` الموجود في إعدادات Vercel.

النشر يعتمد على `vercel.json` لتحديد بناء `@vercel/node` للـ API و`@vercel/static` للصفحة الثابتة.

جاهز للانطلاق؟

انسخ المشروع إلى GitHub، اربطه بـ Vercel، ثم أضف متغير البيئة `ANTHROPIC_API_KEY`. بعد ذلك سيتم تشغيل الصفحة مباشرةً.

يمكنك تعديل النصوص والعناوين بسرعة لتخصيص الصفحة لأي منتج أو تطبيق AI.
